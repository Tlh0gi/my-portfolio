My Portfolio
A modern, responsive portfolio website built with Next.js to showcase my projects, skills, and experience as a developer.
Features

Responsive Design - Works seamlessly on desktop, tablet, and mobile devices
Project Showcase - Display your best work with descriptions and links
Skills Section - Highlight your technical skills and expertise
About Me - Share your story and background
Contact Form - Let visitors reach out directly through the site
Google Maps Integration - Show your location with an embedded map
Admin Dashboard - Manage your content without touching code
Database Integration - Powered by Supabase for real-time data management

Tech Stack

Framework: Next.js 14
Styling: CSS3, Custom Stylesheets
Database: Supabase
Deployment: Vercel
Maps: Google Maps Embed API
Icons: React Icons

Getting Started
Prerequisites

Node.js 18+ installed
npm or yarn package manager
Supabase account
Google Maps API key
